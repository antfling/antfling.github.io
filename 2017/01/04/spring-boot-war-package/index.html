关于spring boot项目无法在8.5.0.1版本过滤器无法正常注册的原因

1.在was 8.5.0.1上 spring boot无法找到SpringServletContainerInitializer是WebApplicationInitializer接口的实现,所以无法初始化过滤器

2.解决方法是在was的server的jvm参数那里加上-Dclassinfocachesize=16000即可

3.如果还不行，请将spring boot的启动类实现WebApplicationInitializer接口即可
